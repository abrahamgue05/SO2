FTP

Por Eduardo Abraham López Guerrero

Definición

  • Protocolo de Transferencia de Archivos (FTP)
  • Protocolo de red estándar utilizado para la transferencia de archivos
  • A través de una red basada en TCP
  • Basado en la arquitectura cliente-servidor

Cliente FTP

Software que se instala en la máquina del usuario y permite conectar mediante FTP a un servidor FTP

Servidor FTP

Software que corre en los servidores conectados a Internet (o LAN) y su función es que el cliente pueda conectarse a él

Funcionamiento

  • Abre dos conexiones que vinculan las computadoras que intentan comunicarse
  • Una conexión es para comandos/respuestas, la otra maneja la transferencia de datos

Títulos en HTML

Por Eduardo Abraham López Guerrero

Definición

  • Los encabezados implementan seis niveles en el documento
  • h1 es el más importante y h6 es el menos importante
  • Describen brevemente el tema de la sección que presentan

Niveles de encabezado

  • <h1> define el encabezado más importante.
  • <h6> define el encabezado menos importante.
  • Van del h1 al h6, siendo el h1 el más grande y el h6 el más pequeño.

Ejemplos


								<h1>Encabezado 1</h1>
								<h2>Encabezado 2</h2>
								<h3>Encabezado 3</h3>
								<h4>Encabezado 4</h4>
								<h5>Encabezado 5</h5>
								<h6>Encabezado 6</h6>

PHP

Por Eduardo Abraham López Guerrero

Definición

  • PHP es un lenguaje de programación de código abierto.
  • Diseñado para el desarrollo web del lado del servidor (backend).
  • Permite crear sitios dinámicos incrustando su código en HTML.

Características

  • Interpretado: No requiere compilación previa.
  • Lado del servidor: Se ejecuta antes de enviar la respuesta al usuario.
  • Tipado dinámico: Las variables no requieren declarar estrictamente su tipo.
  • Soporte de BDD: Integración nativa con MySQL, PostgreSQL, etc.

Ventajas

  • Gratuito: Al ser de código abierto, reduce los costos de desarrollo.
  • Comunidad: Excelente documentación oficial y recursos abundantes.
  • Multiplataforma: Funciona en Windows, Linux y macOS.
  • Rendimiento: Sus versiones recientes (PHP 7 y 8) son muy rápidas.

Aplicaciones

  1. Sistemas CMS: Motor de plataformas como WordPress y Joomla.
  2. Comercio Electrónico: Tiendas en línea con Magento o WooCommerce.
  3. Frameworks Web: Desarrollo avanzado usando Laravel o Symfony.
  4. Redes Sociales: Las bases de Facebook fueron escritas en PHP.
  5. Aplicaciones empresariales: Sistemas de gestión empresarial (ERP, CRM, etc).
