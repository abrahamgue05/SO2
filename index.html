FTP

Por Eduardo Abraham López Guerrero

Definición

  • Protocolo de Transferencia de Archivos (FTP)
  • Protocolo de red estándar utilizado para la transferencia de archivos
  • A través de una red basada en TCP
  • Basado en la arquitectura cliente-servidor

Cliente FTP

Software que se instala en la máquina del usuario y permite conectar mediante FTP a un servidor FTP

Servidor FTP

Software que corre en los servidores conectados a Internet (o LAN) y su función es que el cliente pueda conectarse a él

Funcionamiento

  • Abre dos conexiones que vinculan las computadoras que intentan comunicarse
  • Una conexión es para comandos/respuestas, la otra maneja la transferencia de datos

Títulos en HTML

Por Eduardo Abraham López Guerrero

Definición

  • Los encabezados implementan seis niveles en el documento
  • h1 es el más importante y h6 es el menos importante
  • Describen brevemente el tema de la sección que presentan

Niveles de encabezado

  • <h1> define el encabezado más importante.
  • <h6> define el encabezado menos importante.
  • Van del h1 al h6, siendo el h1 el más grande y el h6 el más pequeño.

Ejemplos


                <h1>Encabezado 1</h1>
                <h2>Encabezado 2</h2>
                <h3>Encabezado 3</h3>
                <h4>Encabezado 4</h4>
                <h5>Encabezado 5</h5>
                <h6>Encabezado 6</h6>

PHP

Por Eduardo Abraham López Guerrero

Definición

  • PHP es un lenguaje de programación de código abierto.
  • Diseñado para el desarrollo web del lado del servidor (backend).
  • Permite crear sitios dinámicos incrustando su código en HTML.

Características

  • Interpretado: No requiere compilación previa.
  • Lado del servidor: Se ejecuta antes de enviar la respuesta al usuario.
  • Tipado dinámico: Las variables no requieren declarar estrictamente su tipo.
  • Soporte de BDD: Integración nativa con MySQL, PostgreSQL, etc.

Ventajas

  • Gratuito: Al ser de código abierto, reduce los costos de desarrollo.
  • Comunidad: Excelente documentación oficial y recursos abundantes.
  • Multiplataforma: Funciona en Windows, Linux y macOS.
  • Rendimiento: Sus versiones recientes (PHP 7 y 8) son muy rápidas.

Aplicaciones

  1. Sistemas CMS: Motor de plataformas como WordPress y Joomla.
  2. Comercio Electrónico: Tiendas en línea con Magento o WooCommerce.
  3. Frameworks Web: Desarrollo avanzado usando Laravel o Symfony.
  4. Redes Sociales: Las bases de Facebook fueron escritas en PHP.
  5. Aplicaciones empresariales: Sistemas de gestión empresarial (ERP, CRM, etc).

Python

Por Eduardo Abraham López Guerrero

Definición

  • Lenguaje de programación de alto nivel y propósito general.
  • Creado por Guido van Rossum y lanzado en 1991.
  • Su filosofía de diseño enfatiza la legibilidad y claridad del código.

Características

  • Sintaxis clara: Utiliza la indentación para delimitar bloques de código.
  • Multiparadigma: Soporta orientación a objetos, programación imperativa y funcional.
  • Interpretado y dinámico: Se ejecuta línea por línea y no declara tipos.
  • Librería estándar: Incluye una enorme cantidad de herramientas por defecto.

Ventajas

  • Fácil de aprender: Curva de aprendizaje muy suave para principiantes.
  • Versatilidad: Útil desde desarrollo web hasta inteligencia artificial.
  • Comunidad gigante: Infinidad de tutoriales, foros y librerías de terceros (PyPI).
  • Alta productividad: Permite escribir menos código para tareas complejas.

5 Aplicaciones

  1. Ciencia de Datos e IA: Machine Learning con Pandas y TensorFlow.
  2. Desarrollo Web (Backend): Creación de servidores y APIs con Django o Flask.
  3. Automatización y Scripting: Tareas repetitivas y web scraping.
  4. Aplicaciones de Escritorio: Interfaces gráficas usando Tkinter o PyQt.
  5. Finanzas y Análisis: Procesamiento de datos financieros y criptográficos.

Java

Por Eduardo Abraham López Guerrero

Definición

  • Lenguaje de programación orientado a objetos y de alto nivel.
  • Fue desarrollado originalmente por Sun Microsystems en 1995.
  • Sigue la filosofía "WORA": Escribe una vez, ejecuta en cualquier lugar.
  • Diseñado para tener la menor cantidad posible de dependencias de implementación.

Características

  • Independiente de plataforma: Su código se compila en bytecode ejecutado por la JVM.
  • Orientado a objetos: Todo se basa en clases y objetos, facilitando el mantenimiento.
  • Seguro: Entorno de ejecución cerrado con un estricto manejo de memoria.
  • Multihilo: Permite ejecutar múltiples tareas de forma simultánea.

Ventajas

  • Portabilidad extrema: Si un equipo tiene la JVM, el programa funcionará sin cambios.
  • Robusto: Verificación estricta de errores y recolector de basura (Garbage Collector).
  • Escalable: Altamente eficiente para soportar aplicaciones que crecen rápidamente.
  • Ecosistema maduro: Cuenta con frameworks muy sólidos y de nivel empresarial.

5 Aplicaciones

  1. Desarrollo Móvil: Ha sido el lenguaje nativo principal para crear apps en Android.
  2. Sistemas Empresariales: Usado masivamente en servidores, bancos y corporaciones.
  3. Big Data: Tecnologías de procesamiento masivo (como Apache Hadoop) lo utilizan.
  4. Internet de las Cosas (IoT): Utilizado en dispositivos integrados, smart TVs y sensores.
  5. Herramientas de Software: Creador de entornos de desarrollo (IDE) como Eclipse o NetBeans.

C

Por Eduardo Abraham López Guerrero

Definición

  • Lenguaje de programación de propósito general y de nivel medio.
  • Creado por Dennis Ritchie en los Laboratorios Bell en 1972.
  • Diseñado originalmente para desarrollar el sistema operativo Unix.
  • Base e inspiración directa para muchos lenguajes modernos (C++, Java, C#).

Características

  • Compilado: El código fuente se traduce directamente a lenguaje máquina.
  • Estructurado: Basado en funciones y bloques lógicos de control (if, while, for).
  • Fuertemente tipado: Requiere la declaración explícita de tipos de datos.
  • Acceso a bajo nivel: Permite la manipulación directa de la memoria mediante punteros.

Ventajas

  • Velocidad extrema: Es uno de los lenguajes más rápidos y eficientes de ejecución.
  • Control total: Otorga al programador gestión manual de memoria y recursos del hardware.
  • Portabilidad: Un programa en C puede compilarse en casi cualquier arquitectura existente.
  • Fundacional: Aprender C facilita enormemente entender cómo funcionan las computadoras.

5 Aplicaciones

  1. Sistemas Operativos: Gran parte del núcleo de Windows, Linux y macOS está escrito en C.
  2. Sistemas Embebidos: Programación de microcontroladores, electrodomésticos y automóviles.
  3. Desarrollo de Compiladores: Lenguajes como Python o PHP están implementados utilizando C.
  4. Motores de Bases de Datos: Sistemas robustos como MySQL o PostgreSQL utilizan C en su núcleo.
  5. Motores de Videojuegos: Creación de software gráfico donde el rendimiento es absolutamente crítico.

JavaScript

Por Eduardo Abraham López Guerrero

Definición

  • Lenguaje de programación interpretado, orientado a objetos y ligero.
  • Conocido principalmente por ser el lenguaje universal de la web.
  • Creado por Brendan Eich en 1995.
  • Permite implementar funciones complejas e interactivas en páginas web.

Características

  • Multiparadigma: Soporta programación orientada a objetos, imperativa y funcional.
  • Interpretado / JIT: Usa compilación Just-In-Time en los navegadores modernos.
  • Tipado dinámico: Las variables no se asocian a un tipo de dato estricto.
  • Asíncrono: Basado en un ciclo de eventos (Event Loop) para manejar múltiples tareas.

Ventajas

  • Ejecución en el cliente: Reduce la carga del servidor al ejecutarse directo en el navegador.
  • Ecosistema gigante: Cuenta con NPM, el registro de software más grande del mundo.
  • Full-Stack: Permite programar tanto el Frontend como el Backend (gracias a Node.js).
  • Interactividad: Mejora enormemente la experiencia del usuario con interfaces dinámicas.

5 Aplicaciones

  1. Desarrollo Web Frontend: Interfaces interactivas con librerías como React, Angular o Vue.js.
  2. Desarrollo Web Backend: Creación de servidores rápidos y APIs RESTful utilizando Node.js.
  3. Aplicaciones Móviles: Desarrollo de apps nativas para iOS y Android mediante React Native.
  4. Aplicaciones de Escritorio: Software multiplataforma (como VS Code o Discord) usando Electron.
  5. Juegos Web: Creación de videojuegos interactivos de navegador usando HTML5 Canvas y WebGL.
